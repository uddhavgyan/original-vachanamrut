# Original Vachanamrut

The goal of this project is to make the original and pure Vachanamrut of Lord Shri Swaminarayan (Shriji Maharaj) available in digital form along with the Sanskrit commentary (Setumala).

**Website:** [https://uddhavgyan.github.io/original-vachanamrut/](https://uddhavgyan.github.io/original-vachanamrut/)  
**GitHub Repository:** [https://github.com/uddhavgyan/original-vachanamrut](https://github.com/uddhavgyan/original-vachanamrut)

## About the Project

The Vachanamrut is the primary scripture of the Swaminarayan Sampradaya, containing 262 discourses of Shriji Maharaj. Over time, changes occurred in manuscripts, so to preserve the original purity:

- Sadguru Shatanand Muni translated the original Gujarati Vachanamrut into Sanskrit, creating the text **Shriharivakyasudhasindhu**.
- The first Acharya of the Vadtal Gadi, Shri Raghuvirji Maharaj, wrote an extensive Sanskrit commentary on it called **Setumala**.

This project presents the original Sanskrit text, Setumala commentary, and translations in Gujarati, Hindi, and English together. The website is built using the Hugo static site generator, and all content is stored in Markdown files within the `content/` folder.

## How to Contribute

Your contributions to this open-source project are most welcome! Help is especially needed in the following areas:

### Main Types of Contributions
- Verification and corrections of the original Sanskrit text and Setumala commentary.
- **Gujarati**, **Hindi**, or **English** translations of the Setumala commentary.
- Linguistic improvements or error corrections in existing translations.
- Adding content for incomplete Vachanamruts.
- Improvements to the website's design, user experience, or new features (CSS, Hugo themes, JavaScript, etc.).

### Contribution Process (Step-by-Step)
1. **Fork** this repository on GitHub.
2. **Clone** it to your computer:

`git clone https://github.com/your-username/original-vachanamrut.git`

3. Create a new **branch** (use a meaningful name, e.g., `tika-g1-english`):

`git checkout -b your-branch-name`

4. Edit the relevant Vachanamrut Markdown file in the `content/` folder.
- Always keep Sanskrit text in **Devanagari** script.
- Translations must be pure, accurate, and respectful.
- If possible, mention the source (original book name/page).

5. **Commit** your changes (with a clear message):

`git add .`
`git commit -m "Gadhada Pratham 1: Added English translation of Setumala commentary"`

6. **Push** your branch and open a **Pull Request** to the main repository.
7. In the Pull Request, provide a detailed description of what changes were made and why.

### Important Guidelines
- This is a religious scripture, so always maintain the purity and respect of the text.
- For large or potentially controversial changes, discuss in GitHub Issues first.
- You can preview the website locally:

`hugo server`
